スマートSEO 修正整形ツール

顧客の修正要望を「同じルール・同じ型」で整理してから手打ちするためのツール。反映漏れ・「伝わらない/AIっぽい」を無くします。

STEP 1顧客メッセージからAIで整形（任意）

顧客の生メッセージを貼って「AI変換プロンプトをコピー」→ 共有アカウントのAI（Claude Code / Codex / GPTs）に貼り付けると、下の型に沿った下書きが返ります。その結果をSTEP2に転記。※AIを使わず直接STEP2に書いてもOK。

→ 共有AIアカウントに貼るだけ。ルールは全員共通です。

STEP 2修正を型に整理（1修正＝1ブロック）

要望の数だけブロックを追加。「対象箇所」は見出し名や該当文を引用で具体的に。事実系（数値・創業年・実績）は根拠を確認（捏造禁止）。トーン系はAfterに文例を1つ。

STEP 3整形して出力 → 手打ちへ

標準フォーマットで出力します。手打ち前に「反映チェック（要望数＝ブロック数）」を確認。
